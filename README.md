# My Git Repo

My GitHub portfolio made with ReactJs

### Libraries and tools used

- [React-Router-Dom]
- [React-icons]
- [axios]
- [prop-type]

## Installation
```bash
npm install react-router-dom
```
```bash
npm install react-icons
```
```bash
npm install axios
```
```bash
npm install prop-types
```
